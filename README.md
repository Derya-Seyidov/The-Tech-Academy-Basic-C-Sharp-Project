## Projects

### Car Insurance

This project is an ASP.NET MVC web application that simulates a car insurance website. Users can enter their personal and vehicle information, and the application calculates an insurance quote based on predefined business rules.

The project includes:
- MVC architecture
- Entity Framework Code First database setup
- Data models and validation
- CRUD functionality
- Admin page for viewing submitted insurance quotes


### Student Database (Entity Framework Code First)

This project demonstrates creating a database using Entity Framework Code First. A Student database was created with C# models and migrations, and student records were added through the application.

The project includes:
- Entity Framework Code First approach
- Database migrations
- Model creation
- Data management using C#
